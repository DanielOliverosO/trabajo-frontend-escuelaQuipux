Trabajo Frontend Daniel Oliveros

Este proyecto es la parte front end de una aplicaciónweb desarrollada con Angular para la gestión de estudiantes, permitiendo registrar, consultar y administrar información académica de los estudiantes de Quipux.

Requerimientos

Requisitos del sistema
Node.js (recomendado v18+)
npm (recomendado v9+)

Dependencias principales

Angular 20.x
Angular Material
FontAwesome para Angular
RxJS
